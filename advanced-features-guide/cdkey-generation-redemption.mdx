---
title: "CDKey Generation & Redemption"
---

What is a CDKey?

A CDKey is a unique redemption code that can be used to:

·Allocate proxy resources (e.g., 100GB traffic)

·Resell proxy services (set custom pricing for clients)

How to Generate a CDKey?

Step 1: Purchase a Premium Plan

·Upgrade to Premium to unlock CDKey features.

Step 2: Access CDKey Dashboard

·Go to Dashboard → CDKey → Generate.

Step 3: Configure CDKey

Traffic Quota Set allocation (e.g., 100GB).

Expiry Date Default: 30 days.

Generated CDKey quantitye (e.g., 5).

Step 4: Generate

·Click 「Generate」 → System creates unique CDKey (e.g., 7XK9-B2T4-L6P8).

·Bulk Generate: Export multiple CDKeys as CSV.

How to Redeem a CDKey?

1. Log in to B2Proxy → Navigate to 「Redeem CDKey」.

 

2. Enter the CDKey → Click 「Redeem」.

3. Traffic are automatically deposited into the account.

Manage CDKeys

·Track Status: View used/unused keys in 「CDKey Details」.

·Revoke: Click 「Revoke」 for compromised CDKeys.

Important Notes

·Premium Plan Required to generate CDKeys.

·One-Time Use: Each CDKey can be redeemed only once.

·Non-Refundable: Resources cannot be returned after redemption.
