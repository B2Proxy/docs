I. Overview

The Sub-User feature allows main accounts to allocate proxy resources (traffic) to team members with controlled permissions, ideal for enterprise or team collaboration.

II. Step-by-Step Instructions

1. Create Sub-User

1.1. Log in as main account → Navigate to 「Sub-Users」.

1.2. Click 「Add Sub-User」.

1.3. Fill in:

·Username (Required): For proxy authentication (e.g., team_dev1).

·Password (Required): Use a strong password (e.g., A1b2@C3d4!).

·Traffic Limit: Set quota (e.g., 10GB).

·Notes (Optional): Label purpose (e.g., "Marketing Team").

1.4. Click 「Confirm」.

Note:

The default main user (sp) cannot be deleted—only password/quota can be modified.

2. Edit Sub-User

·Update Quota/Password: Click 「Edit」 → Modify → Submit.

·Enable/Disable: Toggle status (gray = disabled).

3. Delete Sub-User

·Click 「Delete」 → Confirm.

·Deleted sub-user quotas return to the main account.

III. FAQ

Q1: Can sub-users access the main dashboard?

·No. Sub-users only authenticate proxy connections—no dashboard access.

Q2: How to prevent password leaks?

·Reset passwords regularly.

·Set strict traffic limits to minimize risks.
