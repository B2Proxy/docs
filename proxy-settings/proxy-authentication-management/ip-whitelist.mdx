I. Overview

The IP Whitelist is a security feature that restricts proxy access to only authorized IP addresses, preventing unauthorized usage.

II. Step-by-Step Instructions

1. Add Whitelisted IPs

1.1. Log in → Go to 「Whitelist」 section.

1.2. Click 「Add Whitelist」.

1.3. Enter:

·IP Address (Required): IPv4 format only (e.g., 123.123.123.123).

·Notes (Optional): Label the purpose (e.g., "Finance Dept Server").

1.4. Click 「Confirm」.

Important:

·Only whitelist your private IPs (e.g., office/home networks).

·Do not add public server IPs (security risk).

2. Edit Whitelist

·Single IP: Click 「Notes」 → Edit → Submit.

·Bulk Edit: Select multiple IPs → Click 「Batch Notes」 → Update.

3. Delete Whitelist

·Single IP: Click 「Delete」.

·Bulk Delete: Select IPs → Click 「Batch Delete」.
