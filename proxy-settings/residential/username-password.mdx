## I. Basic Setup Steps

### 1. Select Proxy Location

·Country/Region: Choose from the list (e.g., USA, Japan).

·State/City (Optional): Refine location (e.g., Los Angeles, California).

### 2. Set Session Type

·Rotating Session: Auto-rotates IP per request (ideal for scraping/data collection).

·Sticky Session: Fixed IP for 5 minutes (default; suits account management).

### 3. Optimize Server Connection

·System recommends the nearest server based on your IP.

·Manually select alternative nodes if needed.

### 4. Bind Sub-User (Required)

·Select an existing sub-user or create one to assign proxy permissions.

II. Proxy Generation & Testing

### 5. Test Connectivity (Optional)

Run in Command Prompt:

curl -x host:port -U "username:password" [ipinfo.io](http://ipinfo.io)

(Replace with your credentials)

### 6. Select Output Format

Proxy details include:

·Host

·Port

·Username

·Password

### 7. Set Proxy Quantity

Enter the number of IPs needed (e.g., 100) → Click 「Generate」.

### 8. Retrieve Proxy List

After generation:

Copy individual proxies from 「Proxy List」.

Export as TXT/CSV for bulk use.

<Warning>
  Security Recommendations

  1. Never share proxy credentials; reset passwords periodically.
  2. Assign sub-user permissions judiciously.
</Warning>
