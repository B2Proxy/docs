B2Proxy allows you to **purchase proxy IPs by region**. The workflow is streamlined, enabling fast access to **static residential proxy** services without complicated configurations. This article covers the full process of selecting IPs by region, retrieving proxy credentials and completing setup on B2Proxy to help you get started quickly.

## I. Purchase Proxy IPs

Select your preferred proxy IP type, choose the target country or region, set the IP validity period and purchase quantity. After confirming all configurations, click Buy Now and complete payment.

## II. Retrieve Proxy Information

Navigate to Dashboard \> Static Residential Proxies \> Purchase by Region. In the IP list, copy the generated proxy parameters and configure them in any application or tool.

Proxy information generally includes: Hostname, Port, Username, Password
