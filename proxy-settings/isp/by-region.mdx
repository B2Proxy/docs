B2Proxy allows you to **purchase proxy IPs by region**. The workflow is streamlined, enabling fast access to **static residential proxy** services without complicated configurations. This article covers the full process of selecting IPs by region, retrieving proxy credentials and completing setup on B2Proxy to help you get started quickly.

## I. Purchase Proxy IPs

You can select your preferred proxy IP type, target country/continent, IP plan, and purchase quantity. After confirming the configuration, click Order Now and complete the payment.

## II. Retrieve Proxy Information

Go to Dashboard → Static Residential Proxies → By Region page, and copy the generated proxy params from the IP list. This information usually includes: Hostname, Port, Username, and Password, which can be directly configured in any application or tool.

If you need to configure the IP on a mobile device, go to IP list → Actions → QR code, then scan the QR code with the APP to quickly set up the proxy.
