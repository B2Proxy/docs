B2Proxy's static residential proxies support flexible renewal with a simple process, allowing you to continue using the original IP without reconfiguration. This article will guide you on how to quickly complete the renewal process for static residential proxy IPs, ensuring uninterrupted proxy service.

## I. Feature Description

When automatic renewal is enabled, the system will automatically deduct the fee from your wallet balance 3 days before the IP expires to prevent service interruption.

## II. Enable Auto-Renewal

### Method 1: Batch Setup via IP List

1. Log in → Navigate to 「ISP Proxies」 \> 「IP List」.
2. Select IPs to renew (multiple allowed).
3. Click 「Auto-Renew」 (or 「Batch Renew」 for bulk operations).
4. Confirm details → Click 「Submit」.

Success: Status turns blue.

### Method 2: Individual Setup via Renewal Page

1. Click 「Renew」 next to the target IP.
2. In the payment window, toggle 「Auto-Renew」 to ON (blue).
3. Click 「Continue Payment」 to complete.

## III. Disable Auto-Renewal

1. Locate IPs with active auto-renewal.
2. Click the 「ON」 status hover box → Select 「OFF」 (status turns black).

<Note>
  Note: Must be done at least 3 days before the next billing cycle.
</Note>

## IV. Key Notes

Payment: Deducted only from wallet balance. Ensure sufficient funds.

Failure Alerts: If payment fails, an email reminder will be sent.

Manual Renewal: If automatic renewal is turned off, manual renewal is required. If the IP has already expired, renewal will not be possible.
