## 一、如何参与推荐计划？

### 1. 注册成为会员

登录 B2Proxy官网 并完成注册。

### 2. 获取专属推荐链接/推广码

登录后进入 「仪表板」→「推广计划」

复制您的 唯一推荐链接 或 推广码

### 3. 邀请好友注册并消费

好友通过您的链接或推广码注册

好友购买 任意公开套餐（定制除外）

### 4. 赚取佣金

好友每笔成功订单（套餐购买或钱包充值），您将获得 3%~10% 佣金

佣金以 钱包余额 形式自动发放

## 二、佣金计算规则

好友累计消费金额（美元） 佣金比例：

| 累计消费金额     | 佣金比例 |
| ---------- | ---- |
| ≤1,000     | 3%   |
| ≤10,000    | 5%   |
| ≤30,000    | 7%   |
| ≥ \$30,000 | 10%  |

## 三、注意事项

### 不参与佣金的情况：

定制计划、企业计划

使用钱包余额支付的订单

好友退款或拒付

通过客服手动创建的订单

### 无限制奖励：

可邀请无限好友

佣金金额无上限

## 四、如何提现佣金？

进入 「钱包」→「余额提现」

选择提现方式（银行卡/PayPal等）

输入金额并提交申请（最低提现额度为 \$50）
