---
title: "如何参加推荐计划？"
---

B2Proxy 推荐计划让您在分享代理服务的同时还能赚取佣金。只需 **登录账号获取专属推荐链接**，将链接分享给朋友、同事或社交圈，当他们通过该链接注册并完成订单后，您就能获得高达 **10% 的返利奖励** 自动计入账户余额。本文将带您一步步了解如何参与推荐计划、分享链接并获取收益。

## 一、如何参与推荐计划？

### 1. 注册成为会员

登录 [B2Proxy官网](https://www.b2proxy.com) 并完成注册。

### 2. 获取专属推广链接/推广码

登录后进入 「仪表盘」→「推广返利」，复制您的 唯一推广链接 或 推广码。

### 3. 邀请好友注册并消费

好友通过您的链接或推广码注册，购买 任意公开套餐（定制除外）。

### 4. 赚取佣金

好友每笔成功订单（套餐购买或钱包充值），您将获得 3%~10% 佣金。

## 二、佣金计算规则

好友累计消费金额（美元） 佣金比例：

| 累计消费金额     | 佣金比例 |
| ---------- | ---- |
| ≤1,000     | 3%   |
| ≤10,000    | 5%   |
| ≤30,000    | 7%   |
| ≥ \$30,000 | 10%  |

## 四、如何提现佣金？

进入 「推广返利」→「提现」

选择提现方式：B2钱包余额 / 虚拟货币

输入金额并提交申请（最低提现额度为 \$50）

## 注意事项：

| 不参与佣金的情况    | 无限制奖励   |
| ----------- | ------- |
| 定制计划、企业计划   | 可邀请无限好友 |
| 使用钱包余额支付的订单 | 佣金金额无上限 |
| 好友退款或拒付     |         |
| 通过客服手动创建的订单 |         |
