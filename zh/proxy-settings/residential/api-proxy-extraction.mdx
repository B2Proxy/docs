## 一、基础设置步骤

### 1. 添加IP白名单

进入「IP白名单」

注：未添加白名单的IP将无法调用API

### 2. 选择代理地理位置

国家/地区：从下拉列表中选择目标国家（如美国、日本）

### 3. 设置代理数量

输入需要的IP数量（例如：100个）

### 4. 协议与格式配置

协议：默认HTTP(S)，可选SOCKS5

格式：默认TXT（每行一个IP:端口），可选JSON

分隔符：默认回车换行

### 5. 生成API URL

点击「生成URL」按钮

系统返回唯一API链接

## 二、调用API获取代理

① 直接访问URL

在浏览器或工具中打开生成的API链接，返回示例：

1.1.1.1:8888

2.2.2.2:8888

...

② 编程调用

<Warning>
  频率限制：1秒/次
</Warning>

<Warning>
  代理有效期：默认24小时，超时后需重新获取
</Warning>
