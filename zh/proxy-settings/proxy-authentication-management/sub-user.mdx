## 一、功能概述

子账户功能允许主账户分配代理资源（流量）给团队成员，并控制其使用权限，适合企业或团队协作。

## 二、操作步骤

### 1.创建子账户

登录主账户 → 进入 「子账户」 页面

点击 「新增子账户」

填写以下信息：

用户名（必填）：用于连接代理

密码（必填）：建议使用强密码

流量上限：例如 10GB

备注（选填）：标注用途

点击 「确认」

<Warning>
  注意：系统默认生成的主账户不可删除，仅能修改密码或配额。
</Warning>

### 2.编辑子账户

修改配额/密码：点击子账户右侧的 「编辑」 按钮 → 更新信息 → 提交

启用/禁用：在编辑页面切换状态（禁用为灰色）

### 3.删除子账户

点击子账户右侧的 「删除」 按钮 → 确认操作

## 三、常见问题

### Q1: 子账户能否登录主账户仪表板？

不能，子账户仅用于代理连接认证，无仪表板访问权限。

### Q2: 如何防止子账户密码泄露？

定期修改密码，并限制配额以减少风险。
