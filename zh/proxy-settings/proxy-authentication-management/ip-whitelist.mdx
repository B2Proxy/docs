---
title: "白名单IP"
---

IP 白名单是 B2Proxy 提供的一种安全验证方式，通过仅允许已授权的 IP 地址访问代理服务，来有效防止未授权连接。

在使用 B2Proxy 的 API 提取代理时，您只需在仪表盘中添加需要使用代理的本地或服务器 IP，即可完成白名单配置。配置完成后，只有白名单内的 IP 才能成功提取和使用代理。

本教程将引导您完成白名单 IP 的添加与管理，帮助您快速、安全地使用代理服务。

## 一. 添加白名单IP

1. 登录仪表板 → 进入 「白名单IP」 页面
2. 点击 「添加白名单」，**填写以下信息：**

- IP地址（必填）：仅支持IPv4格式（如 123.123.123.123）
- 备注（选填）：标识IP用途（如“公司服务器-财务部”）

3. 点击 「确认」 完成添加

<Warning>
  注意事项：仅添加您独享的IP（如公司/家庭网络IP），勿添加公共服务器IP。
</Warning>

## 二. 编辑白名单

1. 单个编辑：点击IP右侧的 「编辑」 图标 → 修改IP地址和备注 → 确认
2. 批量备注：勾选多个IP → 点击 「批量备注」 → 统一修改备注

## 三. 删除白名单

1. 单个删除：点击IP右侧的 「删除」 按钮
2. 批量删除：勾选多个IP → 点击 「批量删除」
